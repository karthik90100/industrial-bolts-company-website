# Bolt Manufacturing Company Website

A modern industrial website built for a **bolt manufacturing company** to showcase its products, company information, and services with a clean and trustworthy user interface.

This project focuses on creating a **professional industrial website** that clearly communicates product quality, company credibility, and technical information.

---

# Project Overview

This website is designed to help manufacturing companies present their products in a professional way and build trust with potential clients.

The website includes:

• Detailed company introduction
• Product showcase for bolts and industrial fasteners
• Manufacturing capabilities and facilities
• Industry certifications
• Contact and enquiry system
• Smooth animations for better user experience

The goal of this project is to create a **clean, user-friendly, and modern industrial website**.

# Installation

Clone the repository

git clone https://github.com/your-username/bolt-manufacturing-website.git

Navigate to the project directory

cd bolt-manufacturing-website

Install dependencies

npm install

Start development server

npm run dev

The website will run locally at

http://localhost:5173

---

# Build for Production

To create a production build run

npm run build

This generates an optimized production version of the website.
